# Dream Escape

A narrative-driven puzzle game developed as part of the final-year Game Development Project (COMP3150/3151) at Macquarie University.

## About the Game

Dream Escape is a single-player puzzle adventure inspired by escape room experiences. Players journey through dream-like environments that represent fragments of the protagonist Lily's memories. By solving environmental puzzles and uncovering clues, players gradually piece together a story that is intentionally left open to interpretation.

The game combines:

* Narrative-focused puzzle design
* Environmental storytelling
* Exploration-based gameplay
* Immersive visual and audio experiences

## Key Features

* Multiple puzzle types including logic and environmental interaction puzzles
* Story progression through exploration and puzzle completion
* Atmospheric environments inspired by memory fragments
* Interactive object highlighting and player guidance systems
* Narrative rewards tied directly to puzzle completion
